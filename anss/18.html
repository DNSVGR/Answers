Оператор условного перехода switch. Примеры использования 
                Формат:
                switch(выражение-переключатель)
               {
               case const.выражение 1: оператор 1 ; [break;]
               case const.выражение 2: оператор 2 ; [break;]
               …………………………………………………
               case const.выражение i : оператор i ; [break;]
               …………………………………………………..
               case const.выражение n: оператор n ; [break;]
               [default оператор n+1;]
               } 
               Оператор break используется для выхода из switch. Выражение-переключатель – это выражение или переменная, которая может иметь значение равные i константным выражениям.
                Константные выражения i – это конкретные значения.  
               Оператор i – один или группа операторов.
               Выполнение оператора: вычисляется значение выражения-переключатель, если это значение совпадает с одним из константных выражений, то выполняется оператор, соответствующий этому константному выражению. Если в выражении case отсутствует оператор break, то будут выполнены последующие case, до тех пор, пока не встретится оператор break, либо список case будет окончен. Если break присутствует в выбранном операторе case, то осуществляется выход из оператора switch. Если значение выражения не совпало ни с одним константным выражением, то выполняется оператор, стоящий после default.
