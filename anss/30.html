Инициализация указателей 
                1.	Присваивание указателю адреса существующего объекта или переменной:
                •	с помощью операции взятия адреса
                          int *p=&a;  или  int *p(&a);
                •	с помощью значения другого указателя
                          int *d=p;
                •	с помощью имени массива
                          int B[20];
                          int *k=B;
                •	указатель на функцию может быть проинициализирован именем существующей функции
                
                void f(int x);
                          void (*put) (int);
                put=f;
                2.	Присвоение указателю адреса области памяти в явном виде
                           char *v=0x88000000;
                3.	Присвоение пустого значения указателю
                           int *s=NULL;
                           int *s=0;
                4.	Присвоение указателю адреса, выделенного участка динамической памяти
