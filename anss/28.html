Понятие указателя. Виды указателей. Их описание. 
            int *a
            Есть три вида указателей:
            1.	на объект или переменную
            2.	на функцию
            3.	на тип void
                     Описание приведенных видов указателей.
            1.	тип *имя
                     Тип указателя задается по типу переменной, адрес которой хранится в указателе.
             int *a;
             char *k;
                    Объявление указателя 
                 int *a;
            говорит о том, что значение, записанное по адресу a имеет тип int
            
                  2.   тип (*имя) (список аргументов)
             float (*fun) (float x, float y);
             
            3.  void *имя
                      Этот тип используется, когда тип переменной, адрес которой будет храниться в указателе, не определен, когда он станет известным, используется операция явного преобразования типа. 
              Никогда не следует писать знак * слитно с типом данных: 
            int* a,b;. 
            В этой строке создается впечатление о том, что описаны два указателя на тип int, в то время как на самом деле описан один указатель a на int и одна переменная b типа int.
