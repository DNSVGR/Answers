Связь одномерных массивов с указателями. Динамическое выделение памяти для одномерных массивов. 
                Прототипы функций malloc(), calloc(), функции free() находятся в заголовочном файле alloc.h
                Прототип функции malloc()
                
                void* malloc(unsigned k);
                
                Выполнение
                Функция возвращает адрес выделенной памяти в байт.
                Можно сначала описать указатели, а потом выделить память.
                int *a; char *c;
                a = (int*) malloc(40);  
                c=(char*)malloc(100);
                
                Прототип функции calloc()
                
                void* calloc(unsigned n,unsigned m);
                
                Выполнение
                Функция возвращает адрес выделенной памяти для размещения n элементов по m байт каждый.
                
                Примеры:
                int *a = (int*) сalloc(20,sizeof(int));  //выделяется память под 20 элементов
                char *c=(char*)calloc(100,1);   // выделяется память под 100 элементов (символов)
                float *b=(float*) calloc(10,4); // выделяется память под 10 элементов
                
                В случае невозможности выделения памяти функции возвращают значение NULL.
                Рекомендуется делать проверку: выделена память или нет.
                Освобождение памяти делается с помощью функции free()
                Прототип функции
                void free(void *a);
            
                Указатель – это переменная и y=a; или y++; допустимые операции. Имя массива – константа. Поэтому инструкции a++ и z=&a использовать нельзя.
